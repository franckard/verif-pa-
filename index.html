❓ Questions fréquentes

Rien ne se passe quand je clique sur le favori ?
Vérifiez que vous êtes sur facturation.chorus-pro.gouv.fr/annuaire après avoir passé le CAPTCHA.
Message "session expirée" ?
Rechargez la page F5, repassez le CAPTCHA, recliquez ⭐.
Mon fichier n'est pas reconnu ?
Il faut une colonne dont l'en-tête contient SIREN avec des numéros à 9 chiffres. Formats : .xlsx .xls .csv.
Quelle API est utilisée ?
L'API interne de l'annuaire Chorus Pro avec votre jeton de session. Aucune donnée ne transite par un serveur tiers — tout reste dans votre navigateur.
