❓ Questions fréquentes

Rien ne se passe quand je clique sur le favori ?
Vérifiez que vous êtes bien sur facturation.chorus-pro.gouv.fr/annuaire après avoir passé le CAPTCHA. Le bookmarklet ne fonctionne que depuis cette page précise.
Message "session expirée" ?
Rechargez la page F5, repassez le CAPTCHA, puis recliquez sur le favori ⭐. La session Chorus Pro expire au bout d'environ 30 minutes d'inactivité.
Le glisser-déposer du bouton ne fonctionne pas ?
Clic droit sur la barre de favoris → Ajouter une page → Nom : Vérif PA → URL : copiez l'attribut href du bouton doré (clic droit → Inspecter).
Mon fichier n'est pas reconnu ?
Le fichier doit contenir une colonne dont l'en-tête contient le mot SIREN (insensible à la casse) avec des numéros à 9 chiffres. Formats supportés : .xlsx .xls .csv.
Quelle API est utilisée ?
L'outil appelle l'API interne de l'annuaire Chorus Pro (/annuaire/annuaire-api/unite-legale/{siren}) avec le jeton Bearer récupéré dans votre session active. Aucune donnée ne transite par un serveur tiers. Tout est traité localement dans votre navigateur.
Le délai entre requêtes est-il important ?
Oui. Le délai par défaut est de 300 ms pour éviter un bannissement temporaire par l'API Chorus Pro. Ne descendez pas en dessous de 200 ms. Vous pouvez l'augmenter si vous observez des erreurs (ex. 429 Too Many Requests).
